# Criando-um-resumo-utilizando-o-aplicativo-NotebookLM-do-google.
Entrega do projeto de resumo do livro Principios - Ray Dálio utilizando o NotbookLM para realizar tarefas de sintetização de forma simples, compacta e de compreensão inteligente.
Foram utilizados links do youtube que eram partes do livro Princípios de Ray Dálio, onde ele descreve como conseguiu tirar sua empresa próxima da falência até se tornar uma das maiores empresas de investimentos da atualidade.
Foi utilizado também um arquivo em pdf do livro Princípios, em conjunto com o arquivo de video que se encontrava divididos em 3 partes.
